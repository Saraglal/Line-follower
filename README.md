# Line-follower
The line follower robot is a mobile machine that can detect and follow the line drawn on the floor, implemented with Arduino UNO.

## Components
- two IR sensors
- two gear motors
- H-bridge motor driver (L293D)
- 9v battery
- arduino uno
- breadboard
- wires
